Features

Everything you need for NARDL analysis

A complete, production-ready implementation of the CGS (2019) and SYG (2014) frameworks.

Two-Step Estimation

FM-OLS / FM-TOLS long-run equation + OLS ECM short-run dynamics, exactly replicating Cho, Greenwood-Nimmo & Shin (2019).

PSS Bounds Test

Complete critical value tables for all 5 PSS deterministic cases and k = 0…10 regressors, with automatic decision rules.

Wald Asymmetry Tests

Automatic Wald tests for long-run, short-run additive, and short-run impact asymmetry with χ² statistics.

Dynamic Multipliers

Cumulative dynamic multipliers for positive and negative shocks, with asymmetry profile and LR convergence.

Lag Selection

AIC/BIC grid search with full or partial automation: leave any lag as None and it will be selected optimally.

Publication Charts

Matplotlib-based charts: multiplier plots, CUSUM stability, IRFs, persistence profiles, and residual diagnostics.

Pandas Native

Works directly on DataFrames with DatetimeIndex for natural time-series workflows. Results are ported as attributes.

Stata-Faithful Port

Numerically validated against the original Stata v3.0.0 package. All HAC, FM-OLS, and CUSUM computations match.

Quick Start

From data to results in minutes

from twostep_nardl import TwoStepNARDL

# Estimate Two-Step NARDL with FM-OLS long-run step
model = TwoStepNARDL(
    data      = df,           # pd.DataFrame with DatetimeIndex
    depvar    = 'lcpi',      # dependent variable
    xvars     = ['loil'],    # regressors
    decompose = ['loil'],    # asymmetric decomposition
    lags      = None,         # auto-select via BIC
    maxlags   = 6,
    method    = 'twostep',   # 'twostep' or 'onestep'
    step1     = 'fmols',     # FM-OLS long-run estimator
    case      = 3,            # PSS case III (unrestricted intercept)
    ic        = 'bic',
)

res = model.fit()
print(res.summary())   # publication-quality table

from twostep_nardl.postestimation import bounds_test

# PSS (2001) bounds test for cointegration
bt = bounds_test(res, sig_levels=[0.10, 0.05, 0.01])

# Output:
# ========================================
#   Pesaran, Shin and Smith (2001) Bounds Test
# ========================================
#   F-statistic (PSS)   =      24.3108
#   t-statistic (BDM)   =     -10.0842
#   Case  = 3   k  = 1
#
#   Level   F decision   t decision   Overall
#     10%   REJECT       REJECT       REJECT
#      5%   REJECT       REJECT       REJECT
#      1%   REJECT       REJECT       REJECT

# Access CV table programmatically
from twostep_nardl import pss_cv_table
df_cv = pss_cv_table(case=3, stat='F')

from twostep_nardl.postestimation import multipliers
from twostep_nardl.plotting      import plot_multipliers

# Compute cumulative dynamic multipliers (48 horizons)
mult = multipliers(res, horizon=48)

# mult is a DataFrame with columns: pos, neg, diff
print(mult.head(10))

# Publication-quality plot
plot_multipliers(res, horizon=48)

# Half-life and persistence profile
from twostep_nardl.postestimation import half_life
hl = half_life(res, horizon=72)
print(f"Half-life: {hl['half_life']:.1f} periods")

from twostep_nardl.postestimation import (
    diagnostics, wald_test, ecm_table
)

# Residual diagnostics (BG, White, JB, RESET)
diag = diagnostics(res)

# Wald tests for asymmetry
wt = wald_test(res)

# Publication-style ECM table (CGS 2019 format)
ecm_table(res)

# CUSUM stability test plot
from twostep_nardl.plotting import plot_cusum
plot_cusum(res)

# Impulse response functions
from twostep_nardl.postestimation import irf
irf_df = irf(res, horizon=36)

Methods

Econometric framework

Step 1 — Long-Run (FM-OLS)

ln CPIt = α + β+ Xt+ + β− Xt− + ut

where Xt+ = Σ max(ΔXs, 0) and Xt− = Σ min(ΔXs, 0) are the partial sum processes

  • FM-OLS removes endogeneity bias and serial correlation via non-parametric correction (Andrews-Monahan kernel)
  • FM-TOLS used for multiple asymmetric regressors (k > 1)
  • OLS option available for comparison

Step 2 — Short-Run ECM (OLS)

Δln CPIt = ρ ût−1 + Σ φj Δln CPIt−j + Σ (πj+ ΔXt−j+ + πj− ΔXt−j−) + εt

ρ = speed of adjustment (BDM t-test on ût−1), PSS F-test on level vars

  • Lag structure selected by AIC or BIC grid search
  • HC-robust variance-covariance matrix for all SR parameters
  • Newey-West HAC bandwidth: floor(T0.25) by default
