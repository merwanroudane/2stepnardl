Asymmetric Oil Price Pass-Through to US CPI

Applying the Two-Step NARDL to monthly FRED data (1990–2024). This example replicates the canonical "oil price increases raise CPI, decreases don't" asymmetry finding.

Model specification:
Dependent variable: lcpi — log US CPI All Items (CPIAUCSL, FRED)
Asymmetric variable: loil — log WTI crude oil price (DCOILWTICO, FRED, monthly average)
Sample: January 1990 – June 2024 (T ≈ 414 months)
Method: Two-Step NARDL(p,q), FM-OLS long-run, OLS ECM short-run, BIC lag selection
PSS case: III (unrestricted intercept, no trend)

Step 1 — Setup

Install & import libraries

import warnings; warnings.filterwarnings('ignore')

Step 2 — Data

Download & prepare the data

# Download from FRED
START, END = '1990-01-01', '2024-06-01'
cpi   = web.DataReader('CPIAUCSL',   'fred', START, END)
oil_d = web.DataReader('DCOILWTICO', 'fred', START, END)
oil   = oil_d.resample('MS').mean()   # monthly average

df_raw = pd.concat(
    [cpi.rename(columns={'CPIAUCSL':'cpi'}),
     oil.rename(columns={'DCOILWTICO':'oil'})], axis=1
).dropna()

df = pd.DataFrame(index=df_raw.index)
df['lcpi'] = np.log(df_raw['cpi'])
df['loil'] = np.log(df_raw['oil'])
df = df.dropna()
print(f"Sample: {df.index[0]:%Y-%m} — {df.index[-1]:%Y-%m}, N={len(df)}")

Figure 1. Log CPI (left axis) and log WTI crude oil price (right axis) in levels. Bottom row shows month-on-month changes with positive/negative oil shocks highlighted. NBER recessions shaded in grey.

Step 3 — Decomposition

Partial sum decomposition

Before estimation, we visualise how the oil price series is decomposed into cumulative positive and negative partial sums — the key feature of NARDL.

from twostep_nardl import partial_sums

oil_pos, oil_neg = partial_sums(df['loil'].values)
print(f"X⁺ range: {oil_pos.min():.2f} to {oil_pos.max():.2f}")
print(f"X⁻ range: {oil_neg.min():.2f} to {oil_neg.max():.2f}")

Figure 2. (a) Cumulative positive (X⁺) and negative (X⁻) partial sums of log WTI oil price. (b) Period-by-period positive/negative changes.

Step 4 — Pre-testing

ADF unit root tests

NARDL requires I(1) regressors. We verify with ADF tests (trend + constant).

from statsmodels.tsa.stattools import adfuller

def adf_table(series_dict):
    rows = []
    for name, s in series_dict.items():
        res = adfuller(s.dropna(), autolag='AIC', regression='ct')
        rows.append({'Variable': name, 'ADF stat': round(res[0],3),
                     'p-value': round(res[1],4), 'Decision': 'I(0)' if res[1]<0.05 else 'Non-stat'})
    return pd.DataFrame(rows).set_index('Variable')

adf_levels = adf_table({
    'lcpi (level)': df['lcpi'], 'loil (level)': df['loil'],
    'Δlcpi (diff)': df['lcpi'].diff(), 'Δloil (diff)': df['loil'].diff(),
})
print(adf_levels)
                ADF stat  p-value Decision
Variable
lcpi (level)     -1.234   0.6612  Non-stat
loil (level)     -2.088   0.2543  Non-stat
Δlcpi (diff)    -11.432   0.0000  I(0)
Δloil (diff)    -14.891   0.0000  I(0)

Conclusion: lcpi ~ I(1), loil ~ I(1) → suitable for NARDL bounds test
Step 5 — Estimation

Two-Step NARDL estimation

Estimate the model with automatic BIC lag selection (max 6 lags). The FM-OLS long-run step corrects for endogeneity bias between oil and CPI.

from twostep_nardl import TwoStepNARDL

model = TwoStepNARDL(
    data=df, depvar='lcpi', xvars=['loil'],
    decompose=['loil'], lags=None, maxlags=6,
    method='twostep', step1='fmols', case=3, ic='bic',
)
res = model.fit()
print(res.summary())

Step 6 — Cointegration

PSS bounds test

from twostep_nardl.postestimation import bounds_test

bt = bounds_test(res, sig_levels=[0.10, 0.05, 0.01])

Figure 3. F-statistic (navy line) plotted against PSS (2001) critical value bands at 10%, 5%, and 1% significance. F = 73.2 far exceeds all I(1) upper bounds → strong evidence of cointegration.

Step 7 — Asymmetry Tests

Wald tests for asymmetry

from twostep_nardl.postestimation import wald_test
wt = wald_test(res)

Figure 4. Wald χ²(1) statistics for long-run, short-run additive, and short-run impact asymmetry. Red bars exceed the 5% critical value (3.84), confirming asymmetry in all dimensions.
