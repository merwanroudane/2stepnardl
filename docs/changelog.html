Changelog

All notable changes to twostep-nardl. Follows Semantic Versioning.

v3.0.0 April 2026 Latest
newFull Python port of Stata twostep_nardl v3.0.0 — complete numerical parity with Stata implementation.
newTwoStepNARDL class with both method='twostep' (FM-OLS + OLS ECM) and method='onestep' (single OLS).
newFM-OLS estimator with Andrews-Monahan non-parametric HAC correction and automatic bandwidth (floor(T^0.25)).
newFM-TOLS estimator for multiple asymmetric regressors (k > 1).
newAIC/BIC grid search lag selection with partial automation (mix of fixed and auto lags).
newPSS (2001) bounds test — complete critical value tables for all 5 cases, k=0…10, F and t statistics.
newWald tests for long-run, short-run additive, and short-run impact asymmetry.
newCumulative dynamic multipliers (multipliers()).
newHalf-life and persistence profile of ECM (half_life()): closed-form formula and Pesaran & Shin (1996) PP(h).
newPeriod-by-period impulse response functions (irf()).
newAsymmetric adjustment speed analysis (asymadj()).
newResidual diagnostics: Breusch-Godfrey, White, Jarque-Bera, RESET (diagnostics()).
newPublication-quality ECM table in CGS (2019) Table 9 format (ecm_table()).
newPrediction and fitted values (predict()): xb, residuals, ECT series.
newCUSUM structural stability plot (plot_cusum()).
newPartial sum decomposition with optional dead-band threshold support.
newHC-robust VCE for all short-run parameters.
newComplete oil price → CPI example notebook with FRED data download and 15 sections.
newFull GitHub Pages documentation site with API reference, worked example, and gallery.
newPublished to PyPI: pip install twostep-nardl.
Stata v3.0.0 2023 (original)
refOriginal Stata implementation by Dr Merwan Roudane. This Python package is a faithful port of all numerical routines.
refStata commands: twostep_nardl, twostep_nardl_estat, twostep_nardl_plot.
