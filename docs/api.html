TwoStepNARDL

Main estimation class. Implements the two-step FM-OLS + OLS estimator (Cho, Greenwood-Nimmo & Shin, 2019) and the one-step OLS approach (Shin, Yu & Greenwood-Nimmo, 2014).

from twostep_nardl import TwoStepNARDL model = TwoStepNARDL( data, depvar, xvars, decompose, lags=None, maxlags=4, ic='bic', method='twostep', step1='fmols', threshold=0.0, bwidth=0, case=3, trendvar=None, restricted=False, exog=None, level=0.95 ) res = model.fit()

Constructor Parameters

ParameterTypeDefaultDescription
datapd.DataFrame—Time-series dataset. Must have a DatetimeIndex or integer index.
depvarstr—Name of the dependent variable column.
xvarslist[str]—Names of all independent variables. Must include all decompose variables.
decomposelist[str]—Subset of xvars to decompose into positive and negative partial sums.
lagsint | list | NoneNoneFixed lag structure [p, q₁, …, qₖ]. Use None for automatic BIC/AIC selection, or list with None entries for partial selection.
maxlagsint | list[int]4Maximum lag considered during grid search. Can be variable-specific as a list.
icstr'bic'Information criterion for lag selection: 'bic' or 'aic'.
methodstr'twostep''twostep' (FM-OLS long-run + OLS short-run) or 'onestep' (single OLS).
step1str'fmols'First-step estimator: 'fmols', 'ols', 'fmtols' (auto for k>1), 'tols'.
thresholdfloat | list0.0Dead-band threshold for partial sum decomposition (per variable).
bwidthint0HAC bandwidth. 0 = automatic (floor(T^0.25)), positive = fixed.
caseint3PSS deterministic case 1–5:
1 = no det., 2 = restricted const., 3 = unrestricted const., 4 = restricted trend, 5 = unrestricted trend.
trendvarstr | NoneNoneColumn name providing the trend variable (required for cases 4 and 5).
restrictedboolFalseIf True, restrict deterministics to the long-run equation only (cases 2, 4).
exoglist[str] | NoneNoneAdditional strictly exogenous regressors (included in SR only).
levelfloat0.95Confidence level for interval construction.

bounds_test()

from twostep_nardl.postestimation import bounds_test bt = bounds_test(res, sig_levels=[0.10, 0.05, 0.01], show_table=True)

Pesaran, Shin & Smith (2001) bounds test for cointegration. Prints a formatted table and returns a dict.

Returns

dict with keys: F_pss, t_bdm, rho, case, k, decisions (dict mapping significance level → 'reject' / 'no decision' / 'inconclusive').

wald_test()

from twostep_nardl.postestimation import wald_test wt = wald_test(res, show_table=True)

Wald tests for long-run, short-run additive, and short-run impact asymmetry.

Returns

dict with keys: 'Long-run symmetry', 'Short-run symmetry (additive)', 'Short-run symmetry (impact)' → each a dict with W and p.

diagnostics()

from twostep_nardl.postestimation import diagnostics diag = diagnostics(res, show_table=True)

Residual diagnostics. Runs four tests on the SR ECM residuals:

  • Breusch-Godfrey — serial correlation (χ² statistic)
  • White — heteroskedasticity
  • Jarque-Bera — normality of residuals
  • RESET — functional form misspecification (Ramsey)
Returns

dict with keys: bg_chi2, bg_p, white_chi2, white_p, jb_stat, jb_p, reset_F, reset_p.

multipliers()

from twostep_nardl.postestimation import multipliers mult = multipliers(res, horizon=40, show_table=True)

Compute cumulative dynamic multipliers for a unit shock to the asymmetric regressors. For k=1, returns a DataFrame with columns pos, neg, diff.

ParameterTypeDescription
horizonintNumber of periods for multiplier computation. Default 40.
show_tableboolIf True, prints a formatted table. Default True.
Returns

pd.DataFrame with index 1…H and columns pos, neg, diff (for k=1) or {var}_pos, {var}_neg, {var}_diff per variable.

half_life()

from twostep_nardl.postestimation import half_life hl = half_life(res, horizon=40, show_table=True)

Compute the half-life and persistence profile of the ECM. Based on the closed-form formula HL = −ln(2)/ln(1+ρ) and the Pesaran & Shin (1996) persistence profile.

Returns

dict with keys: rho, rho_se, rho_t, rho_p, half_life, mal (mean adjustment lag), full_adj (99% adjustment time), pp_halflife, pp_series (pd.Series of PP(h) values).

irf()

from twostep_nardl.postestimation import irf irf_df = irf(res, horizon=20, show_table=True)

Period-by-period impulse response functions derived from the cumulative dynamic multipliers.

Returns

pd.DataFrame with columns: pos_period, neg_period, pos_cum, neg_cum.

asymadj()

from twostep_nardl.postestimation import asymadj adj = asymadj(res, horizon=40, show_table=True)

Asymmetric adjustment speed analysis: compares the long-run impact of positive vs. negative shocks and reports the shared ECM half-life.

ecm_table()

from twostep_nardl.postestimation import ecm_table ecm_table(res)

Display a publication-quality ECM parameter table in the style of Table 9 in Cho, Greenwood-Nimmo & Shin (2019). Prints Panel A (long-run) and Panel B (short-run dynamics) with significance stars.

partial_sums()

from twostep_nardl import partial_sums x_pos, x_neg = partial_sums(x, threshold=0.0)

Decompose a series into positive and negative partial sum processes. With the default threshold of 0, this implements:

X⁺_t = Σ_{s=1}^{t} max(Δx_s, threshold) X⁻_t = Σ_{s=1}^{t} min(Δx_s, threshold)

Non-zero thresholds give a "dead-band" (neutral zone) decomposition.

ParameterTypeDescription
xnp.ndarrayInput series (levels).
thresholdfloatDead-band threshold. Default 0.
Returns

Tuple (x_pos, x_neg) — two np.ndarray of same length as input.
