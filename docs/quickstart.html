Quick Start

From install to results in minutes

Follow these steps to run your first Two-Step NARDL estimation using the oil price → CPI example.

1

Install the package

Install twostep-nardl from PyPI. Python 3.9+ required.

pip install twostep-nardl

For the data-download example, also install:

pip install pandas-datareader yfinance
ℹ️ Dependencies (numpy, pandas, scipy, statsmodels, matplotlib) are installed automatically.
2

Load your data

Provide a pd.DataFrame with a time index. The example uses monthly WTI crude oil prices and US CPI from FRED (1990–2024).

import numpy as np
import pandas as pd
import pandas_datareader.data as web

START, END = '1990-01-01', '2024-06-01'
cpi   = web.DataReader('CPIAUCSL', 'fred', START, END)
oil_d = web.DataReader('DCOILWTICO', 'fred', START, END)
oil   = oil_d.resample('MS').mean()

df_raw = pd.concat([cpi.rename(columns={'CPIAUCSL':'cpi'}),
                    oil.rename(columns={'DCOILWTICO':'oil'})], axis=1).dropna()

df = pd.DataFrame(index=df_raw.index)
df['lcpi'] = np.log(df_raw['cpi'])
df['loil'] = np.log(df_raw['oil'])
df = df.dropna()
print(df.tail())
3

Estimate the Two-Step NARDL

Initialise TwoStepNARDL and call .fit(). Set lags=None for automatic BIC lag selection.

from twostep_nardl import TwoStepNARDL

model = TwoStepNARDL(
    data      = df,
    depvar    = 'lcpi',
    xvars     = ['loil'],
    decompose = ['loil'],   # partial-sum decomposition
    lags      = None,         # BIC auto-selection
    maxlags   = 6,
    method    = 'twostep',   # two-step FM-OLS
    step1     = 'fmols',
    case      = 3,            # PSS Case III
    ic        = 'bic',
)
res = model.fit()
print(res.summary())
💡 Use method='onestep' to switch to the single-equation OLS approach (Shin, Yu & Greenwood-Nimmo 2014).
4

Run the PSS bounds test

Test for cointegration using the Pesaran, Shin & Smith (2001) bounds procedure.

from twostep_nardl.postestimation import bounds_test

bt = bounds_test(res, sig_levels=[0.10, 0.05, 0.01])

# Access decisions programmatically:
print(bt['decisions'][0.05])   # 'reject' or 'no decision'
print(f"F = {bt['F_pss']:.3f},  t = {bt['t_bdm']:.3f}")
5

Test for asymmetry (Wald tests)

Test whether the long-run and short-run responses are asymmetric.

from twostep_nardl.postestimation import wald_test

wt = wald_test(res)

# Direct access to stored attributes:
print(f"W_LR = {res.W_lr:.3f} (p = {res.p_lr:.4f})")
print(f"W_SR = {res.W_sr:.3f} (p = {res.p_sr:.4f})")
6

Compute dynamic multipliers

Compute and plot cumulative dynamic multipliers for positive and negative shocks.

from twostep_nardl.postestimation import multipliers
from twostep_nardl.plotting      import plot_multipliers

mult = multipliers(res, horizon=48)  # 4-year horizon
plot_multipliers(res, horizon=48)

# Half-life of adjustment
from twostep_nardl.postestimation import half_life
hl = half_life(res, horizon=72)
print(f"Half-life: {hl['half_life']:.1f} months")
7

Run residual diagnostics

Check serial correlation, heteroskedasticity, normality, and functional form.

from twostep_nardl.postestimation import diagnostics
diag = diagnostics(res)

# CUSUM structural stability
from twostep_nardl.plotting import plot_cusum
plot_cusum(res)

# Publication-style ECM table
from twostep_nardl.postestimation import ecm_table
ecm_table(res)
8

Access results attributes

All estimates are stored as attributes of the NARDLResults object.

# Long-run estimates
print(f"β+ = {float(res.b_lr[0]):.4f}")
print(f"β- = {float(res.b_lr[1]):.4f}")
print(f"ρ  = {res.rho:.4f}")

# As a pandas Series
print(res.coef)
print(res.pvalues)

# ECT series (for custom analysis)
ect = res.ect_series
